# Auth + Dashboard App (Frontend Intern Assignment)

## Tech Stack
- Frontend: React.js
- Backend: Node.js, Express
- Database: MongoDB
- Auth: JWT

## Features
- User signup & login
- JWT-based authentication
- Protected dashboard
- CRUD operations for tasks
- Basic validation & error handling

## Project Structure
- /backend – Express backend APIs
- /frontend – React frontend UI

## How to run (optional)
Backend:
- npm install
- node server.js

Frontend:
- npm install
- npm start

## Scaling (Production)
- Environment variables for secrets
- Refresh tokens
- API pagination
- DB indexing
- Docker + CI/CD
